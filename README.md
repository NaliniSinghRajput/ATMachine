# ATMachine
Hey there! today i made an ATM Machine!, it's very easy to use and is an interactive menu mode.

So like my other begineer projects, this one also includes, if and else loops along with the elif.
I made this ATM, user friendly by putting the options for th euser as a menu mode.

firstly i ask the user how much they wish to have in the account. the user then gives the first deposit, which is the initial balance that my code will further run on.
Next i , let the user decide what want to do with the balance.
i gave them 3 options, 
1. They can check their balance.
2. They can Withdraw money from the account.
   Here, i made three further conditions, using if and else and if statements.
   
   Firstly, if the user wishes to withdraw more money , than in the balance, they will get a message saying that their funds are insufficient.
   Secondly, if the user gives the same withdraw amaount as the balancce that is already there, then the computer tells them that their current balance is now 0.
   Lastly, if the user gives the withdrawl amount, less than the balance, then it will subtract the inputted withdrawl money from the balance, and then it also tells the balance that is newly formed.
   
3. They can deposit money to their account.
